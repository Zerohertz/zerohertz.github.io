레퍼런스

변수에 별명을 붙여서 접근하는 것

  • 형 &으로 선언 해준다
  • 포인터는 변수이지만 레퍼런스는 자신만의 메모리 공간이 할당되지 않음

오류나는 유형

  • 레퍼런스가 가리키는 대상은 변경될 수 없다
  • 선언과 동시에 초기화 시켜야 한다
  • 상수로 초기화 시키면 오류가 난다

name space

Class

  • string이 가장 기본적인 클래스
  • field에서 멤버 변수 선언
  • method에서 멤버 함수 선언
  • public은 접근 지정자로서 클래스 외부에서 멤버를 자유롭게 사용할 수 있음을 의미한다
  • class에 값을 대입하면 안되고 instance에만 값을 대입해야한다

전용 멤버(private member) vs 공용 멤버(public member)

  • 각각 private:, public:과 같이 클래스에서 멤버를 선언하기 전에 명시해둔다
  • 아무것도 없으면 디폴트로 private:과 같이 즉, 전용 멤버로 선언된다

getter와 setter

  • private:선언의 입출력
  • 접근자와 설정자

  • 서로 다른 타입의 변수들을 한곳에 모아 놓은 것
  • 클래스에서 멤버 함수를 제외하면 구조체다
  • 반대로 구조체에 함수를 추가하여 확장한 것이 클래스다
